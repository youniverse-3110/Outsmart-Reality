## 🚀 Deploy to GitHub Pages (Step by Step)

### What You Need
- A free GitHub account: [github.com](https://github.com)
- The file `frequency-tracker.html` from this repository

---

### Step 1 — Create Your GitHub Account
Go to [github.com](https://github.com) → Sign up (free)

---

### Step 2 — Create a New Repository

1. Click **"+"** in the top right → **"New repository"**
2. Repository name: `outsmart-reality`
3. Set visibility to **Public** ✅
4. Check **"Add a README file"** ✅
5. Click **"Create repository"**

---

### Step 3 — Upload the Files

1. In your repository, click **"Add file"** → **"Upload files"**
2. **Rename** `frequency-tracker.html` to **`index.html`** on your computer first
3. Upload both `index.html` and `README.md`
4. Click **"Commit changes"**

---

### Step 4 — Enable GitHub Pages

1. Go to **"Settings"** tab in your repository
2. Scroll to **"Pages"** in the left sidebar
3. Under **"Source"** → select **"Deploy from a branch"**
4. Under **"Branch"** → select **`main`** and **`/ (root)`**
5. Click **"Save"**

---

### Step 5 — Your App is Live 🎉

Wait **1–3 minutes**, then visit:

```
https://YOUR-USERNAME.github.io/outsmart-reality/
```

---

## 🔄 How to Update Your App

When you receive an updated `frequency-tracker.html` file:

1. Go to your repository on GitHub
2. Click on `index.html`
3. Click the **pencil icon** (Edit this file)
4. Select all text → Delete → Paste the new code
5. Click **"Commit changes"**
6. Wait **1–2 minutes** → your live site updates automatically

## ❓ Troubleshooting

| Problem | Fix |
|---------|-----|
| 404 error on the URL | Make sure the file is named exactly `index.html` |
| "Pages" not showing in Settings | Make sure the repository is set to **Public** |
| Changes not appearing after update | Hard refresh: `Ctrl + Shift + R` (Windows) / `Cmd + Shift + R` (Mac) |
| Fonts not loading | The app uses Google Fonts — requires an internet connection |
| Blank white screen | Check that you copied the complete HTML file without truncation |
| iOS not scrolling inside modals | This is handled — tap the × to close and scroll the main page |
